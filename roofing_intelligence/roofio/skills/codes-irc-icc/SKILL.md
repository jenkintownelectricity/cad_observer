# Building Codes - IRC/ICC Skill

## ROOFIO - Division 07 Expert System

### Overview
Building codes establish minimum requirements for roofing installations. The International Code Council (ICC) publishes the International Building Code (IBC) for commercial and the International Residential Code (IRC) for residential construction.

## International Building Code (IBC)

### Chapter 15: Roof Assemblies and Rooftop Structures

#### Section 1501: General
- Roof assemblies shall be designed for applicable loads
- Weather protection required
- Fire classification required (when applicable)

#### Section 1502: Definitions

**Roof Assembly:** System of interacting components including deck, vapor retarder, insulation, and membrane.

**Roof Covering:** The outermost weather-resistant layer.

**Roof Deck:** The structural surface to which roof assembly is applied.

**Built-Up Roof Covering:** Two or more layers of felt with bitumen and surfacing.

**Membrane Roof Covering:** Flexible single or multi-ply sheet characterized by flexibility.

#### Section 1503: Weather Protection

**1503.1 General:**
- Roof decks shall be covered with approved roof coverings
- Fastened to resist wind loads per Chapter 16

**1503.2 Flashing:**
- Shall be installed at:
  - Wall and roof intersections
  - Roof plane changes
  - Wherever required to prevent moisture infiltration

**1503.3 Coping:**
- Parapet walls shall be coped with noncombustible materials
- Minimum 2:12 slope to interior

**1503.4 Roof Drainage:**
- Designed to prevent ponding
- Secondary (overflow) drainage required where primary drainage is internal
- Overflow 2" above low point of roof
- Capacity to prevent water above the roof level

#### Section 1504: Performance Requirements

**1504.1 Wind Resistance:**
- Roof assemblies shall be designed per ASCE 7
- Tested per approved standards (FM 4470, UL 580, etc.)

**1504.3 Uplift Resistance:**
```
Method 1: Tested assembly (FM, UL listed)
Method 2: ASCE 7 C&C pressures
Method 3: Engineering analysis
```

**1504.4 Aggregate:**
- Shall not be used where wind speed exceeds:
  - 105 mph (Exposure B)
  - 95 mph (Exposure C)
  - 90 mph (Exposure D)

**1504.5 Edge Securement:**
- Metal edge flashing tested per ANSI/SPRI ES-1

#### Section 1505: Fire Classification

**Roof Classifications:**
| Class | Fire Test Rating | Description |
|-------|------------------|-------------|
| A | Highest | Effective against severe fire |
| B | Moderate | Effective against moderate fire |
| C | Light | Effective against light fire |

**Testing Standards:**
- ASTM E108: Fire Tests of Roof Coverings
- UL 790: Standard Test Methods for Fire Tests of Roof Coverings

**Where Required:**
- All buildings unless exempt
- Buildings within fire district: Class A or B
- WUI (Wildland-Urban Interface): Special requirements

#### Section 1507: Roof Covering Materials

**1507.1 Scope:**
- Material standards listed by type
- Installation requirements

**1507.10 Built-Up Roofs:**
- Conform to Table 1507.10
- Materials: ASTM D312, D2178, D4601, etc.

**1507.11 Modified Bitumen:**
- Conform to Table 1507.11
- Materials: ASTM D6162, D6163, D6164, etc.

**1507.12 Thermoset (EPDM) Single-Ply:**
- ASTM D4637 Type I, II, or III
- Minimum 45 mil thickness

**1507.13 Thermoplastic (TPO/PVC) Single-Ply:**
- ASTM D6878 (TPO)
- ASTM D4434 (PVC)
- Minimum 45 mil (TPO), 48 mil (PVC)

**1507.14 Sprayed Polyurethane Foam:**
- ASTM C1029
- Protective coating required
- Minimum density: 2.5 pcf

**1507.15 Liquid-Applied Coatings:**
- ASTM C836, C957, D1227, D6083, D6694

#### Section 1510: Reroofing

**1510.1 General:**
- New roof covering shall comply with current code
- Re-cover permits under certain conditions

**1510.3 Recovering vs. Replacement:**
| Condition | Re-cover | Replacement |
|-----------|----------|-------------|
| Existing layers | ≤1 | >1 |
| Structural adequacy | Verified | Required |
| Wet insulation | None | Any present |
| Deteriorated deck | None | Any present |

**1510.4 Roof Replacement:**
- All layers removed to deck
- Deck repaired/replaced as needed
- New assembly per current code

## International Residential Code (IRC)

### Chapter 9: Roof Assemblies

#### Section R903: Weather Protection

**R903.1 General:**
- Roof decks covered with approved materials

**R903.2 Flashing:**
- Required at:
  - Chimneys
  - Walls
  - Roof valleys
  - Roof plane intersections
  - Around openings

**R903.4 Roof Drainage:**
- Designed to prevent damage to adjacent properties
- Gutters/downspouts where required by local code

#### Section R904: Materials

**R904.1 Scope:**
- Same classifications as IBC (Class A, B, C)

**R904.2 Fire Classification:**
- Dwelling units require rated assembly unless:
  - Non-classified allowed by local jurisdiction

**R904.3 Roof Covering Limitations:**
- Wood shakes/shingles restrictions in fire areas

#### Section R905: Requirements by Type

**R905.2 Asphalt Shingles:**
```
Standards: ASTM D225, D3018, D3462
Slope: Minimum 2:12 (4:12 normal)
Underlayment: Required (Type I or II)
Ice barrier: Required in cold climates
```

**R905.4 Metal Roof Shingles:**
- ASTM D3161 (wind resistance)
- Minimum slope: Per manufacturer

**R905.5 Metal Roof Panels:**
- Minimum 26 gauge (steel)
- Underlayment as required
- Slope: Per manufacturer

**R905.7 Wood Shingles:**
- Minimum slope: 3:12
- Spaced sheathing permitted
- Underlayment per climate zone

**R905.8 Wood Shakes:**
- Minimum slope: 3:12
- Interlayment required between courses
- Fire treatment available

**R905.10 Built-Up Roofing (Residential):**
- Same requirements as IBC
- Aggregate restrictions apply

**R905.13 Thermoplastic Single-Ply:**
- ASTM D6878 (TPO)
- Minimum slope: 1/4:12

#### Section R905.1.1: Ice Barriers

**Required in Climate Zones:**
```
Required: Zones 5, 6, 7, 8 and areas with history of ice dams

Location: From eave edge to 24" inside exterior wall line
         (minimum 36" from eave)

Alternative: 2 layers standard underlayment
```

#### Section R908: Reroofing (Residential)

**R908.3 Recovering:**
- Maximum 2 layers total
- Structural adequacy verified
- No wet materials present

**R908.4 Roof Replacement:**
- Remove all layers to deck
- Inspect and repair deck
- New installation per current code

## Special Requirements

### High-Wind Regions

**Enhanced Requirements:**
- Engineered attachments
- Enhanced edge metal (ES-1 compliant)
- Impact-resistant in hurricane zones
- Product approvals (FL, TX, etc.)

### Wildland-Urban Interface (WUI)

**Fire-Resistant Requirements:**
- Class A roof assembly minimum
- No combustible materials on roof
- No vent openings exceeding 1/4"
- Noncombustible gutters

### Historic Buildings

**Code Exceptions:**
- May maintain existing materials
- Alternative compliance methods
- Fire safety systems as alternative
- Documented in permit

## Permit Requirements

### Typical Roofing Permit Requirements

**Documentation:**
- [ ] Permit application
- [ ] Roof plan (existing and proposed)
- [ ] Manufacturer specifications
- [ ] Wind uplift calculations or FM listing
- [ ] Energy compliance documentation
- [ ] Licensed contractor information

**Inspections:**
- Pre-roof (deck condition)
- Insulation/vapor retarder
- Membrane/cover installation
- Final inspection

### Inspection Checklist

**Pre-Installation:**
- [ ] Deck type and condition verified
- [ ] Slope adequate for drainage
- [ ] Penetrations properly installed
- [ ] Substrate ready for roofing

**Final:**
- [ ] Material matches specifications
- [ ] Flashing heights meet code
- [ ] Drainage functional
- [ ] Fire rating verified (if required)
- [ ] Wind attachment verified
