# Drop Folder

Place files here for processing by CAD Observer.

Supported file types:
- Archives (.zip, .rar, .7z)
- Screenshots (.png, .jpg)
- CAD files (.dwg, .dxf)
- Specification documents (.pdf, .docx)
- Reference images
- Command logs (.txt)

**Note:** Most file types in this folder are ignored by git to keep the repository clean. Only this README is tracked.
