# Division 07 Inspections Skill

## ROOFIO - Division 07 Expert System

### Overview
Systematic inspection is critical for quality roofing and waterproofing installations. This skill covers inspection protocols, checklists, and documentation requirements for all phases of roofing projects.

---

## Inspection Types

### Pre-Installation Inspection

**Deck Inspection:**
```
□ Deck type verified against specs
□ Deck properly attached
□ No excessive deflection
□ Surface clean and dry
□ No protruding fasteners
□ Joints tight (<1/4" gaps)
□ Slope adequate for drainage
□ Deck dry (moisture test if needed)
```

**Steel Deck Checklist:**
| Item | Requirement | Check |
|------|-------------|-------|
| Gauge | Per spec (20-22 ga typical) | □ |
| Attachment | Welds or fasteners per FM | □ |
| Flutes | Properly oriented | □ |
| Surface | No oil, rust, debris | □ |
| Openings | Framed and supported | □ |

**Concrete Deck Checklist:**
| Item | Requirement | Check |
|------|-------------|-------|
| Cure | 28 days minimum | □ |
| Moisture | <5% per ASTM D4263 | □ |
| Surface | No laitance, smooth | □ |
| Cracks | Identified, addressed | □ |
| Level | Within tolerance | □ |

**Wood Deck Checklist:**
| Item | Requirement | Check |
|------|-------------|-------|
| Type | Plywood or OSB as spec'd | □ |
| Thickness | Per structural design | □ |
| Gaps | <1/8" between panels | □ |
| Fastening | 6" OC edges, 12" OC field | □ |
| APA stamp | Exposure rating verified | □ |

### Progress Inspections

**Vapor Retarder:**
```
□ Material matches submittal
□ Continuous installation
□ Laps properly sealed
□ Turned up at walls/curbs
□ Sealed at penetrations
□ No tears or punctures
```

**Insulation:**
```
□ Material and thickness verified
□ R-value confirmed
□ Joints offset (6" minimum)
□ Multiple layers staggered
□ No gaps between boards
□ Attachment per wind design
□ No wet or damaged boards
□ Tapered slope correct
```

**Cover Board (if specified):**
```
□ Type per specification
□ Thickness verified
□ Joints staggered from insulation
□ Attachment adequate
□ Surface acceptable for membrane
```

**Membrane:**
```
□ Material matches submittal
□ Correct thickness
□ Proper overlap (side and end)
□ Seams welded/adhered properly
□ 100% seam probe test passed
□ No wrinkles or bridging
□ No fishmouths at intersections
□ Field attachment per design
```

**Flashings:**
```
□ Base flashing height (8" min)
□ Material compatible
□ Properly terminated at top
□ Corners wrapped correctly
□ No bridging at angle changes
□ Sealants properly applied
□ Counter flashing installed
□ Edge metal per ES-1
```

### Final Inspection

**Complete System Checklist:**
```
MEMBRANE:
□ All seams probe tested and passed
□ No punctures or damage
□ Surface clean
□ Walkway pads installed (if required)

FLASHINGS:
□ All flashings complete
□ Heights verified
□ Terminations sealed
□ Counter flashings in place

DRAINAGE:
□ Drains functional
□ Strainers in place
□ Overflow functional
□ No ponding (or documented)

PENETRATIONS:
□ All penetrations flashed
□ Pipe boots sealed
□ Equipment curbs complete
□ Conduit/pipe supports installed

EDGE METAL:
□ Perimeter complete
□ Joints properly sealed
□ Attachment verified
□ Expansion joints functional

ACCESSORIES:
□ Vents installed
□ Hatches functional
□ Fall protection anchors
□ Lightning protection

DOCUMENTATION:
□ Daily reports complete
□ Photos organized
□ Test results documented
□ Warranty paperwork ready
```

## Inspection Procedures

### Daily Inspection Protocol

**Start of Day:**
1. Weather conditions check
2. Review previous day's work
3. Material verification
4. Equipment inspection
5. Safety compliance check

**During Work:**
1. Monitor installation methods
2. Check quality continuously
3. Verify dimensions/thicknesses
4. Document progress (photos)
5. Note any issues immediately

**End of Day:**
1. Review completed work
2. Verify temporary protection
3. Complete daily report
4. Plan next day inspection

### Seam Inspection

**Visual Inspection:**
```
Check for:
□ Consistent seam width
□ No wrinkles in seam area
□ No burn-through (heat weld)
□ Complete edge seal
□ T-patch at intersections
□ No fishmouths
□ Proper alignment
```

**Probe Test Protocol:**
1. Wait for seam to cool (5+ minutes)
2. Use blunt-tipped probe
3. Firm, consistent pressure
4. Test entire length of every seam
5. Mark defects immediately
6. Verify repairs
7. Document results

### Adhesion Inspection

**Adhered Membrane:**
- Check for full contact
- No air pockets
- Adhesive coverage verified
- Bond strength spot-checked

**Mechanically Attached:**
- Fastener pattern correct
- Plates properly seated
- No plate distortion
- Stress-free membrane over plates

## Inspection Standards

### Tolerance Standards

| Item | Tolerance |
|------|-----------|
| Ponding | <48 hrs after rain |
| Flashing height | +0", -0" of spec |
| Seam width | ±1/4" of spec |
| Insulation gaps | <1/4" |
| Edge metal joints | <1/8" gap |

### Acceptance Criteria

**Seams:**
- 100% pass probe test
- Minimum width achieved
- No defects

**Flashings:**
- Height requirements met
- Proper termination
- Sealed as specified

**Overall:**
- No defects visible
- Drainage functional
- All work complete
- Documentation complete

### Red Flags

| Observation | Action |
|-------------|--------|
| Wet insulation | Remove, replace |
| Failed seams | Repair, re-test |
| Low flashings | Correct immediately |
| Ponding water | Document, evaluate |
| Missing fasteners | Add per design |
| Material damage | Replace |
